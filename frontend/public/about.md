# QSOlive

**See your club's QSOs, live**

**Version:** {{VERSION}}

QSOlive is a real-time ham radio contact mapping application. It visualizes ADIF log entries from multiple operators on an interactive map, built for ham radio clubs to track and display contacts as they happen.
